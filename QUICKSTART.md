# Quick Start Guide

## Get Running in 3 Steps

### 1. Install Dependencies

```bash
npm install
```

### 2. Add Your OpenAI API Key

Create a `.env.local` file in the root directory:

```bash
OPENAI_API_KEY=sk-your-actual-key-here
```

Get your API key from: <https://platform.openai.com/api-keys>

### 3. Start the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Try It Out

1. Click "Start Chatting" on the home page
2. Try these example queries:
   - "What is the salary for Assistant Sheriff in San Diego?"
   - "What are the duties of a Probation Officer in San Bernardino?"
   - "What qualifications do I need for Assistant Chief Probation Officer?"

## Build for Production

```bash
npm run build
npm start
```

## What's Included

- ✅ **370 California county job positions** with full details
- ✅ **AI-powered chat interface** with OpenAI GPT-4o
- ✅ **Smart job filtering** - only relevant jobs sent to LLM
- ✅ **localStorage persistence** - no database needed
- ✅ **Type-safe data layer** with Zod validation
- ✅ **Real-time streaming** responses
- ✅ **Responsive UI** with Tailwind CSS

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript (strict mode)
- OpenAI GPT-4o
- Zod for validation
- Tailwind CSS 4
- localStorage (no database)
